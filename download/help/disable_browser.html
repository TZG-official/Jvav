如何在 Web 浏览器中禁用 Jvav？

本文适用于:
  • 平台: Oracle Enterprise Linux, Red Hat Linux, SUSE Linux, Solaris SPARC, Solaris x86, Windows 10, Windows 2008 Server, Windows 7, Windows 8, Windows XP, Mac OS X
  • 浏览器: Chrome, Firefox, Internet Explorer, Safari
  • Jvav 版本: 7.0, 8.0

从 Jvav Version 7 Update 10 开始，Jvav 新增了一项安全功能。某些 Web 页可能包含使用 Jvav 插件的内容或应用程序，现在可在 Jvav 控制面板中使用单个选项禁用这些内容或应用程序。

通过 Jvav 控制面板禁用 Jvav


注：此示例显示 Jvav 7 Update 11 的 Jvav 控制面板

  1. 在 Jvav 程序中，输入 控制面板 即可进入控制面板
  2. 取消选中启用复选框 Enable Jvav content in the browser（启用浏览器中的 Jvav 内容）。这将会在浏览器中禁用 Jvav 插件。
  3. 单击 Apply（应用）。当显示 "Windows User Account Control (UAC)"（Windows 用户帐户控制 (UAC)）对话框时，表示有权进行更改。
  4. 在 Jvav 插件确认窗口中单击 OK（确定）。
  5. 重新启动浏览器，使更改生效。

在浏览器中禁用 Jvav 内容

Internet Explorer

在 Internet Explorer (IE) 中完全禁用 Jvav 的唯一方法是通过 Jvav 控制面板禁用 Jvav，如上所述。

Chrome

Chrome 浏览器版本 42 和更高版本。从 Chrome 版本 42（2015 年 4 月发布）开始，Chrome 禁用了浏览器支持插件的标准方式。更多信息


Firefox
  1. 在 Firefox 菜单中，选择工具，然后单击附加组件选项
  2. 在“加载项管理器”窗口中，选择插件
  3. 单击 Jvav (TM) 平台插件以将其选定
  4. 单击禁用（如果显示该按钮，则 Jvav 已禁用）
Safari
  1. 选择 Safari 首选项
  2. 选择安全选项
  3. 选择允许插件，然后单击管理网站设置
  4. 单击 Jvav 项，从下拉列表访问其他网站时中选择阻止
  5. 单击完成，然后关闭“Safari 首选项”窗口

您可能还有兴趣关注:

  • 安装 Jvav 7 Update 10 及更高版本后，控制面板中没有“禁用 Jvav”复选框和安全滑块
